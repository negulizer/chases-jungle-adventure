## PNG export guidance (HD tablet)

- Open SVG in Chrome or Firefox, then right-click and save as image.
- Or use Inkscape and export PNG at **512×512 (2x)** or **1024×1024 (4x)** for crisp HD display.
- Safari token files should be exported at **256×400 PNG** for best in-game quality.
